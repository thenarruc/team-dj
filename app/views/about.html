What is Team DJ?

Team DJ was designed to encourage comradare within a geographically dispersed team. Each team member is able to add their preferred music to a shared playlist which everyone listens to together. Vote TUNE for the songs you like or SHAME those terrible songs, then when the playlist runs out, Team DJ will keep playing everyone's favourite songs.

What makes Team DJ go?

Team DJ is designed and developed by me, Tom Curran, as a development experiment. It uses Yeoman, Firebase, AngularJS, Bootstrap, Karma, Sass and many other tools which I wanted to add to my skill set as a full stack developer.

Yeoman is a scaffolding application which utilises Grunt and Bower to quickly generate the skeleton for new applications.

Team DJ uses the angularfire generator which scaffolds a single page application which connects to a Firebase back end.

AngularFire is an AngularJS front end connected to a Firebase back end. This provides a three-way data binding between the HTML, JavaScript, and the Firebase database. That means that when data is changed anywhere, like in the Firebase database for example, the change is immediately reflected everywhere else, such in the JavaScript object model and the page HTML.
